Capacity Trend (last 7 days)

Report Key This view tracks the recent SQL snapshots for the current filter scope. The API applies the active region preset, specific region, selected subscriptions, family, and availability filters first, then groups dbo.CapacitySnapshot by day. Total SKU Observations is the number of stored observations for that day, while Total Quota Available is the summed headroom across those observations. Large day-to-day swings usually mean a different number of snapshot rows were captured on that day, not that the React client changed the region filter.
